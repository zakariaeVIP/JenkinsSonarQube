TP CI/CD Pipeline

Mise en place d'un pipeline CI/CD sous Windows 10

1. Préparation du projet Web

Node.js

Installez Node.js depuis nodejs.org et créez un projet :

mkdir my-node-project
cd my-node-project
npm init -y
npm install express

Ajoutez un fichier app.js avec ce contenu :

const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
    res.send('Hello, CI/CD Pipeline!');
});

app.listen(port, () => {
    console.log(`App running at http://localhost:${port}`);
});

PHP

Créez un projet PHP dans un serveur local comme XAMPP :

<?php
echo "Hello, CI/CD Pipeline!";
?>

Ajoutez ce fichier dans htdocs/my-php-project.

2. Création du dépôt GitHub

Créez un dépôt sur GitHub et clonez-le :

git clone https://github.com/votre-utilisateur/ci-cd-project.git

Ajoutez vos fichiers et poussez-les :

git add .
git commit -m "Initial commit"
git push origin main

3. Installation des outils

Jenkins

Téléchargez Jenkins depuis jenkins.io, installez-le, et configurez les plugins nécessaires.

SonarQube

Téléchargez SonarQube depuis sonarqube.org, lancez-le et configurez votre projet.

4. Configuration du pipeline CI/CD

Exemple de Jenkinsfile

Ajoutez ce fichier à la racine de votre projet :

pipeline {
    agent any
    stages {
        stage('Build') {
            steps {
                echo 'Building the application...'
                sh 'npm install'
            }
        }
        stage('Test') {
            steps {
                echo 'Running tests...'
                sh 'npm test'
            }
        }
        stage('Code Analysis') {
            steps {
                echo 'Analyzing code with SonarQube...'
                sh 'sonar-scanner'
            }
        }
        stage('Deploy') {
            steps {
                echo 'Deploying application...'
                sh 'cp -R * C:\\pipeline-prod'
            }
        }
    }
}
